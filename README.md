# Face Recognition App (Flask + OpenCV)

This is a Python Face Recognition project using Flask, OpenCV, and face_recognition.

## Features
- Real-time webcam face detection
- Face matching with known images
- Flask web interface

## Tech Stack
- Python
- Flask
- OpenCV
- face_recognition
- dlib

## How to Run

1. Create virtual environment:

python -m venv .venv

2. Activate:

.venv\Scripts\activate

3. Install dependencies:

pip install -r requirements.txt

4. Run:

python main.py

## Note

This project uses your local webcam, so it must be run locally.
It will NOT work on cloud platforms like Heroku.
